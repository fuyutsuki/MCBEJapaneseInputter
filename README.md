# MCBEJapaneseInputter
Minecraft Windows 10 Editionで発生するチャットバグを回避するツール  
  
## 使用方法
jarを起動してMCBE Win10 Edをプレイするだけ。  
「t」もしくは「Enter」キーを押すとウィンドウが開き、そこのテキストボックスに文字を入力してEnterキーを押すことで自動でチャットが送信されます。  
なお、チャット画面を開いた状態でも同様の操作で開きます。
